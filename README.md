# IEI / TKNFRA Research

Open-access repository of scientific papers produced by the **Intelligence
Economy Institute (IEI)** and **TKNFRA**. Papers are published here as working
papers / preprints prior to, and alongside, submission to peer-reviewed
scientific journals.

## Repository layout

```
.
├── LICENSE                     # CC BY-NC-ND 4.0 (applies to all papers)
├── README.md                   # this file
└── papers/
    └── <year>-<short-slug>/    # one folder per paper
        ├── README.md           # title, authors, abstract, keywords, status, citation
        ├── LICENSE             # paper-level copy of CC BY-NC-ND 4.0
        ├── CITATION.cff        # machine-readable citation metadata
        └── <paper>.pdf         # the manuscript
```

## Adding a new paper

1. Create a folder `papers/<year>-<short-slug>/`.
2. Add the manuscript PDF.
3. Copy the root [LICENSE](LICENSE) into the folder.
4. Add a `README.md` with the title, authors, affiliation, abstract, keywords,
   status, and a citation block (use an existing paper's README as a template).
5. Add a `CITATION.cff` for machine-readable citation metadata.
6. Add a row to the **Papers** table above.
